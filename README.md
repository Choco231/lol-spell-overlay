# 롤 스펠 체크 오버레이

League of Legends 상대 소환사 주문 쿨타임을 화면 위에 띄워 확인하는 Electron 오버레이입니다.

## 실행

처음 실행하는 컴퓨터에서는 아래 파일을 먼저 더블클릭하세요. Node.js가 없으면 `winget`으로 Node.js LTS 설치를 시도한 뒤 의존성을 설치하고 실행합니다.

```text
설치_및_실행.cmd
```

가장 간단한 실행 방법은 아래 파일을 더블클릭하는 것입니다. 콘솔 창 없이 실행됩니다.

```text
롤_스펠_오버레이_실행.vbs
```

문제 원인을 직접 확인하고 싶을 때는 아래 파일을 실행하세요.

```text
롤_스펠_오버레이_실행.cmd
```

처음 실행하는 컴퓨터에서는 `Node.js`와 인터넷 연결이 필요합니다. `node_modules`가 없으면 실행 스크립트가 자동으로 `npm.cmd install`을 시도합니다. Node.js 설치까지 자동으로 처리하고 싶으면 `설치_및_실행.cmd`를 사용하세요.

수동 실행:

```bash
npm.cmd install
npm.cmd start
```

PowerShell에서 `npm`이 실행 정책 때문에 막히면 `npm.cmd`를 사용하세요.

## 조작

- 상단 바 드래그: 오버레이 위치 이동
- `X`: 종료
- `Ctrl + Shift + R`: 오버레이 창 위치와 크기 초기화
- 상단 슬라이더: 오버레이 전체 투명도 조절

## 스펠 사용

- 스펠 아이콘 좌클릭: 쿨타임 시작/취소
- 스펠 아이콘 우클릭: 스펠 변경 창 열기
- 스펠 변경 창에서 아이콘 클릭: 해당 스펠로 변경
- 정글은 강타를 제외하고 스펠 1개만 표시합니다.

## 라인별 보정

각 포지션 아래의 버튼으로 쿨타임 보정을 적용합니다.

- `I`: 아이오니아 장화
- `C`: 우주적 통찰력
- `T`: TOP 전용 강화 텔레포트

## 참고

오버레이 창은 투명 배경, 프레임 제거, 항상 위 표시 옵션으로 생성됩니다. 일부 전체 화면 독점 모드에서는 게임 위에 보이지 않을 수 있으므로 롤은 `테두리 없음` 또는 `창 모드`로 실행하는 편이 안정적입니다.

스펠 이름, 아이콘, 쿨타임은 Riot Data Dragon `16.11.1` 데이터를 기준으로 사용합니다. 네트워크가 막히면 내장 기본 데이터로 동작합니다.
