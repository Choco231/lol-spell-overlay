# 롤 스펠 체크 오버레이

League of Legends 화면 위에 띄워 쓰는 Electron 기반 소환사 주문 쿨타임 오버레이입니다.

## 실행

가장 간단한 실행 방법은 아래 파일을 더블클릭하는 것입니다. 콘솔 창 없이 실행됩니다.

```text
롤_스펠_오버레이_실행.vbs
```

문제 원인을 직접 보고 싶을 때는 아래 파일을 실행하면 됩니다.

```text
롤_스펠_오버레이_실행.cmd
```

의존성이 없으면 처음 실행할 때 자동으로 `npm.cmd install`을 시도합니다.

처음 한 번만 의존성을 설치합니다.

```bash
npm.cmd install
```

실행은 다음 명령을 사용합니다.

```bash
npm.cmd start
```

PowerShell에서 `npm`이 실행 정책 때문에 막히면 `npm.cmd`를 사용하세요.

## 오버레이 조작

- 상단 바를 드래그해서 창 위치 이동
- `게임 클릭` 버튼: 오버레이 클릭을 게임 화면으로 통과시킴
- `Ctrl + Shift + O`: 클릭 통과 모드 켜기/끄기
- `Ctrl + Shift + R`: 오버레이 창 크기와 위치 초기화
- `_`: 최소화
- `X`: 종료

## 스펠 기능

- 상대 5명 라인별 스펠 2개씩 추적
- 스펠 아이콘 클릭으로 쿨타임 시작, 다시 클릭하면 취소
- `변경` 버튼으로 스펠 교체
- 우주적 통찰력, 아이오니아 장화, 강화 텔레포트 보정
- 투명도 조절, 컴팩트 모드, 전체 초기화
- `1`~`5`: 각 라인의 첫 번째 스펠 시작/취소
- `Shift + 1`~`Shift + 5`: 각 라인의 두 번째 스펠 시작/취소
- `Alt + R`: 전체 초기화

## 참고

Electron 창은 `alwaysOnTop`, 투명 배경, 프레임 제거 옵션으로 생성됩니다. 다만 일부 게임의 전체 화면 독점 모드에서는 운영체제 오버레이 창이 게임 위에 보이지 않을 수 있으므로, 롤은 `테두리 없음` 또는 `창 모드`로 실행하는 편이 안정적입니다.

스펠 이름과 아이콘은 Riot Data Dragon URL을 우선 사용하고, 네트워크가 막힌 경우 내장 기본 쿨타임으로 동작합니다.
